Demo: finding the minimum of Himmelblau's function
==================================================
|

Himmelblau's function is defined as

.. math::
   f(x,y) = (x^2 + y - 11)^2 + (x + y^2 - 7)^2

|

Himmelblau's function has four minima at which the function is 0. These minima are located at:

.. math::
    (3, 2)
.. math::
    (-2.805118, 3.131312)
.. math::
    (-3.779310, -3.283186)
.. math::
    (3.584428, -1.848126)

|
